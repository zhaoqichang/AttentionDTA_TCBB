# AttentionDTA_TCBB
 AttentionDTA: drug--target binding affinity prediction by sequence-based deep learning with attention mechanism

This repository contains the source code and the data.

## Setup and dependencies 

Dependencies:
- python 3.6
- pytorch >=1.2
- numpy
- sklearn
- tqdm
- tensorboardX
- prefetch_generator

## Resources:
+ README.md: this file.

	+ KIBA.txt:  
	+ Metz.txt: 
	+ Davis.txt
